### Installation
```bash
$ [sudo] pip install mac-itunes
```

```bash
$ [sudo] npm i -g mac-itunes
```

#### Examples
```bash
$ itunes play
$ itunes pause
$ itunes stop
$ itunes next
$ itunes previous
```

volume
```bash
$ itunes volume 50
$ itunes volume
50
```

mute
```bash
$ itunes mute
$ itunes muted
1
$ itunes unmute
```

frontmost (`1` or `0`)
```bash
$ itunes frontmost
0
```

`iTunes.app` process
```bash
$ itunes pid
42
$ itunes kill
```
